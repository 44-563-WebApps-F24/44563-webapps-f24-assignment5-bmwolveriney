The letters (Y, A, L, L, A, B) use "col-sm-2", giving them a fixed width that keeps them in one row until the screen is very small. In contrast, the dessert names use "col-md" without a set width, allowing them to adjust more easily and wrap to the next line sooner. So, when resizing the window, the dessert names(i.e Apricot Custard Delight) will break first and wrap before the letters do.
